# FTZ Digital Marketing Agency Website

A cutting-edge, fully competitive marketing agency website showcasing 30+ marketing solutions with stunning 3D animations, modern UI/UX, and eye-catching design.

## 🚀 Features

- **3D Interactive Elements**: Powered by Three.js and React Three Fiber
- **30+ Service Solutions**: Comprehensive digital marketing services
- **Modern Animations**: Framer Motion and GSAP for smooth transitions
- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Performance Optimized**: Code splitting, lazy loading, and optimized assets
- **SEO Ready**: Meta tags, semantic HTML, and optimized structure
- **Dark Theme**: Beautiful dark theme with customizable colors
- **AI Chatbot**: Interactive customer support chatbot
- **Theme Toggle**: Dark/light mode support

## 🛠️ Tech Stack

- **Framework**: Next.js 14 with TypeScript
- **3D Graphics**: Three.js, React Three Fiber, @react-three/drei
- **Animations**: Framer Motion, GSAP
- **Styling**: Tailwind CSS
- **Icons**: React Icons
- **Fonts**: Space Grotesk, Poppins, Inter, Orbitron

## 📦 Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
